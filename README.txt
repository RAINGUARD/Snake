Classic snake game programmin in Python. Design elements borrowed from Python Engineer over at https://www.youtube.com/c/PythonEngineer

Author: Neil Haggerty

Running the code:
- Python must be installed
- if on windows, the curses module must be installed
- go to command prompt and type: pip install windows-curses

Once this is complete download snake.py. Double click the file named snake.py to run it.
